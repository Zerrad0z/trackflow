## Navette Maximum Durations

## General Rules
- All dates must be in dd/MM/yyyy format
- Departure time must never be after arrival time
- Station names must be UPPERCASE and match the list above
- ACT agent names must be UPPERCASE
- All text must be free of spelling errors
